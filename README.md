# autobuild.sh
Make DDLC Mod Building as easy as breathing air!

## How to use

Simply `git pull` and run it. It's easy.

Or, if you're just doing it once:

```sh

bash | wget https://raw.githubusercontent.com/Sayo-nika/autobuild.sh/master/autobuild.sh
````
